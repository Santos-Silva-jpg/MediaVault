# MediaVault

## 🇧🇷 Português

Um simples sistema de gerenciamento de mídia escrito em Python.
Ele permite cadastrar, listar, pesquisar e remover itens da sua biblioteca pessoal.

### Funcionalidades

- Adicionar itens
- Listar itens cadastrados
- Pesquisar itens pelo título
- Remover itens
- Salvar dados em um arquivo JSON

### Tecnologias

- Python
- JSON

---

## 🇺🇸 English

MediaVault is a simple media management system developed in Python.

The project allows you to add, list, search, and remove items from your personal library.

### Features

- Add items
- List registered items
- Search items by title
- Remove items
- Save data using a JSON file

### Technologies

- Python
- JSON
